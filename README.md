# Machine Learning Based Buyer Segmentation and Investment Profiling for Real Estate Market Intelligence

## Project Overview

This project uses Machine Learning clustering techniques to identify different buyer segments in the real estate market. The objective is to help real estate companies understand customer behavior, improve marketing strategies, and provide better investment recommendations.

## Problem Statement

Real estate companies often treat all buyers similarly despite significant differences in demographics, investment goals, financing preferences, and geographic locations. This project applies clustering algorithms to discover hidden buyer segments and generate actionable business insights.
